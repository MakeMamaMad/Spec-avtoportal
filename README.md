# Портал «СпецТрейлеры / Рынок и Регламенты» — стартовый проект

Этот репозиторий содержит:
- **frontend/** — статический сайт (HTML/CSS/JS) с уникальным дизайном и тематической сеткой.
- **aggregator/** — Python-ингест новостей/видео/регламентов из RSS и API.
- **.github/workflows/** — GitHub Actions для автосбора и публикации.
- **design/** — гайдлайны по визуальному стилю.
- **legal/** — редакционная политика и ограничения.
- **sql/** — схема БД (если решите перейти на headless CMS).

## Быстрый старт (локально)
1. Откройте `frontend/index.html` в браузере (двойной клик). Для fetch JSON на `file://` лучше запустить простой сервер:
   ```bash
   cd frontend
   python3 -m http.server 8080
   # затем откройте http://localhost:8080
   ```
2. Контент лежит в `frontend/data/*.json` — можно править вручную.
3. Для автосбора установите Python 3.11+, затем:
   ```bash
   cd aggregator
   python3 -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt
   cp .env.example .env  # при необходимости внесите токены
   python main.py
   ```
   Результат выгружается в `../frontend/data/` и дополняет/обновляет ленты.

## Деплой на GitHub Pages (вариант по умолчанию)
1. Создайте репозиторий и запушьте весь проект.
2. Включите GitHub Pages для ветки `main` (корень `/frontend`).
3. Включите GitHub Actions и при необходимости поправьте расписание в `.github/workflows/ingest.yml`.

## Что внутри фронтенда
- Адаптивная сетка, индустриальный стиль, тёмная/светлая темы.
- Лента с подрубриками: *Выставки, Новые модели, Новые производители, Рынок, Уникальные факты, Производители грузовиков/моторов, Дилеры*.
- Раздел «Законодательство / ГОСТы»: фильтры по типам документов.
- Раздел «Гайды и регламенты»: практические руководства.
- Плейлист «Видео»: агрегирование из YouTube/VK/RuTube/Dzen (через RSS/API).
- Места под **нативную рекламу** (с отключателем).

## Правила включения/исключения
- Исключаем из новостной ленты рекламу/упоминания моделей из линейки САТ (зерновозы, панелевозы, контейнеровозы и т.п.) — см. `aggregator/rules.yml`.
- Обязательные темы: выставки, новые модели, новые производители, уникальные/удивительные факты, новости производителей грузовиков и дилеров.
- Разрешено всё остальное (цистерны, стальные прицепы и пр.).
